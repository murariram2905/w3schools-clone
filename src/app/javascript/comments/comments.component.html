JavaScript Comments

Comments are used to explain JavaScript code and make it more readable. They are ignored by the browser and do not affect code execution.

Single-line Comments:

// This is a single-line comment
let x = 5; // Declare a variable

Multi-line Comments:

/*
  This is a multi-line comment
  It can span multiple lines
*/

Use Cases:

  • Explain code logic
  • Temporarily disable parts of code
  • Improve collaboration and maintenance
