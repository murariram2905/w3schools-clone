JavaScript Statements

JavaScript statements are instructions the browser can execute. A program is a list of statements.

Examples:

let x = 5;
let y = 10;
let sum = x + y;
console.log(sum);

document.getElementById("output").innerHTML = "Sum: " + sum;

Click the button to show the result

Tip:

  • Statements usually end with ;
  • Blocks of statements are enclosed in {{ '{' }}...{{ '}' }}
