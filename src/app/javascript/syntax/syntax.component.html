JavaScript Syntax

JavaScript syntax is the set of rules that define a correctly structured JavaScript program.

JavaScript Values:

  • Fixed values: 5, 'Hello' are called literals
  • Variable values: stored in variables using let, var, or const

JavaScript Statements:

let x = 5;
let y = 6;
let z = x + y;

JavaScript is Case Sensitive:

let lastname = "Murari";
let lastName = "Ramakrishna";

// These are two different variables

Key Syntax Rules:

  • Each statement ends with a semicolon ;
  • Blocks of code are written inside {{ '{' }} ... {{ '}' }}
  • JavaScript is case sensitive
