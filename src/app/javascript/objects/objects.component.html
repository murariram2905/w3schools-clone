JavaScript Objects

Objects are used to store keyed collections of various data and more complex entities.

Creating an Object:

const person = {{ '{' }}
  firstName: "Murari",
  lastName: "Ram",
  age: 24,
  fullName: function() {{ '{' }}
    return this.firstName + " " + this.lastName;
  {{ '}' }}
{{ '}' }};

Accessing Object Properties:

person.firstName
person["age"]

Why Use Objects?

  • Group related data and functions
  • Build structured, reusable code
  • Key concept in OOP (Object-Oriented Programming)
