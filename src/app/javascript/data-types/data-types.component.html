JavaScript Data Types

JavaScript provides different types of data that can be used to store values.

Common Data Types:

  • String: text, e.g. "Hello"
  • Number: integer or floating-point, e.g. 100, 3.14
  • Boolean: true or false
  • Array: collection of values, e.g. ["apple", "banana"]
  • Object: key-value pairs, e.g. {{ '{' }}name: "Murari"{{ '}' }}
  • Null: intentionally empty
  • Undefined: variable declared but no value assigned
