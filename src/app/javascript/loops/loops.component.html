JavaScript Loops

Loops can execute a block of code a number of times. Useful when you want to repeat tasks.

Types of Loops:

  • for - loops through a block of code a number of times
  • while - loops through a block of code while a condition is true
  • do...while - also loops while a condition is true, but always runs once first
  • for...of - loops through values of an iterable object like an array

Example - for loop:

for (let i = 0; i < 5; i++) {{ '{' }}
  console.log(i);
{{ '}' }}
