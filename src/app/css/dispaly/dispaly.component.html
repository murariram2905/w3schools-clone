CSS Display

The display property determines how an element is displayed in the document layout.

Common Display Values:

  • block – Takes full width, starts on a new line
  • inline – Flows with text, no width/height support
  • inline-block – Like inline, but supports width and height
  • none – Hides the element from the page

Example CSS:

.block-box {{ '{' }} display: block; background-color: #198754; color: white; margin-bottom: 10px; padding: 10px; {{ '}' }}
.inline-box {{ '{' }} display: inline; background-color: #ffc107; padding: 5px; {{ '}' }}
.inline-block-box {{ '{' }} display: inline-block; background-color: #0dcaf0; padding: 10px; width: 120px; {{ '}' }}

Output:

Block Element
Inline Text

Inline-Block Boxes

Tips:

  • Use inline-block when you want inline layout + box styling
  • display: none hides the element completely
  • Default display: div is block, span is inline
