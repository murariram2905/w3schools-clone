CSS Float

The float property in CSS is used to position elements to the left or right of a container, allowing other content to wrap around them.

Common Float Values:

  • left – Floats the element to the left
  • right – Floats the element to the right
  • none – Default; no floating

Example CSS:

.float-box {{ '{' }}
  width: 150px;
  height: 100px;
  background-color: #198754;
  color: white;
  text-align: center;
  line-height: 100px;
  float: left;
  margin-right: 15px;
{{ '}' }}

.clearfix::after {{ '{' }}
  content: "";
  display: table;
  clear: both;
{{ '}' }}

Output:

Floating Box

This paragraph wraps around the floating box. Float is useful for layouts like image wrapping or side-by-side sections.

Tips:

  • Always clear floats using clear: both or clearfix
  • Float is useful for simple layouts, but Flexbox or Grid is recommended for modern designs
  • Floated elements are removed from the normal document flow
