CSS Comments

Comments in CSS are used to explain the code and make it easier to understand. They are ignored by the browser and do not affect how the page looks.

CSS Comment Syntax:

/* This is a single-line CSS comment */

Example:

p {{ '{' }}
  color: green; /* Set text color */
  font-size: 18px; /* Set font size */
{{ '}' }}

Output:

This paragraph is styled using CSS with comments.

Notes:

  • Comments start with /* and end with */
  • They can be placed on their own line or inline with code
  • Useful for explaining styles or temporarily disabling code
