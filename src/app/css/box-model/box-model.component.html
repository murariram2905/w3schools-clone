CSS Box Model

The CSS Box Model describes how elements are structured and how spacing is calculated around them. Every element is a rectangular box consisting of:

  • Content – the actual text or image
  • Padding – space around the content (inside the border)
  • Border – the edge around the padding
  • Margin – space outside the border

CSS Code:

.box-demo {{ '{' }}
  padding: 20px;
  border: 5px solid #198754;
  margin: 20px;
  background-color: #e8f5e9;
{{ '}' }}

Notes:

  • padding: space between content and border
  • margin: space outside the element
  • border: surrounds the padding and content
  • You can inspect box model using browser dev tools
