CSS Introduction

CSS (Cascading Style Sheets) is used to style and layout web pages — for example, to change fonts, colors, spacing, and element positioning.

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements should be displayed
  • It saves time by controlling the layout of multiple web pages at once
  • It can be used internally, externally, or inline with HTML

Basic Example:

<style>
  p {{ '{' }}
    color: green;
    font-size: 20px;
{{ '}' }}
</style>

Output:

This is a styled paragraph using CSS.

Types of CSS:

  • Inline CSS – inside an HTML element using the style attribute
  • Internal CSS – inside a <style> tag in the <head>
  • External CSS – in a separate .css file

Why Use CSS?

  • Separates content from design
  • Makes pages responsive and accessible
  • Improves maintainability of code
