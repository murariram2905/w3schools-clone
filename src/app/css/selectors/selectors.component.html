CSS Selectors

CSS selectors are used to target HTML elements you want to style.

Types of CSS Selectors:

  • Universal Selector: * targets all elements
  • Element Selector: targets elements by tag name
  • Class Selector: uses a dot . before the class name
  • ID Selector: uses a hash # before the ID
  • Group Selector: targets multiple elements

Example:

* {{ '{' }}
  margin: 0;
{{ '}' }}

p {{ '{' }}
  color: blue;
{{ '}' }}

.highlight {{ '{' }}
  background-color: yellow;
{{ '}' }}

#main {{ '{' }}
  font-weight: bold;
{{ '}' }}

Output:

This is a normal paragraph.

This paragraph has a yellow background (class selector).

This paragraph is bold (ID selector).

Notes:

  • * targets everything (use carefully)
  • p targets all <p> tags
  • .classname is used for reusable styles
  • #idname should be unique per page
