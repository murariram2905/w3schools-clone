CSS Lists

CSS allows you to customize the appearance of lists including bullets, numbers, spacing, and more.

Types of Lists:

  • Unordered List: Uses <ul> with bullets
  • Ordered List: Uses <ol> with numbers

CSS Styling for Lists:

ul.custom-bullets {{ '{' }}
  list-style-type: square;
  margin-left: 20px;
{{ '}' }}

ol.custom-numbers {{ '{' }}
  list-style-type: upper-roman;
  margin-left: 20px;
{{ '}' }}
