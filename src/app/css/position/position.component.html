CSS Position

The position property specifies the type of positioning method used for an element. It can affect how elements are placed in relation to their normal position or their parent.

Position Types:

  • static: Default. Element is placed in normal flow.
  • relative: Positioned relative to its normal position.
  • absolute: Positioned relative to the nearest positioned ancestor.
  • fixed: Positioned relative to the browser window.
  • sticky: Scrolls until a certain position and sticks there.

Example CSS:

.relative-box {{ '{' }}
  position: relative;
  top: 20px;
  left: 20px;
  background-color: #198754;
  color: white;
  width: 200px;
  height: 100px;
  text-align: center;
  line-height: 100px;
{{ '}' }}

Output:

Relative Positioned Box

Tips:

  • position: absolute; removes the element from normal flow
  • position: fixed; is great for headers or navbars
  • relative is useful when nudging an element from its spot
  • Always set top, left, right, or bottom with position types
