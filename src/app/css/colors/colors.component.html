CSS Colors

CSS colors are used to define the color of text, backgrounds, borders, and more. You can specify colors in different formats:

Color Formats:

  • Color Name: e.g., red, blue, green
  • Hex Value: e.g., #ff0000 for red
  • RGB: e.g., rgb(255, 0, 0)
  • RGBA: e.g., rgba(0, 128, 0, 0.5) for transparency
  • HSL: e.g., hsl(120, 100%, 50%)

Example CSS:

.box1 {{ '{' }} background-color: red; {{ '}' }}
.box2 {{ '{' }} background-color: #00ff00; {{ '}' }}
.box3 {{ '{' }} background-color: rgb(0, 0, 255); {{ '}' }}
.box4 {{ '{' }} background-color: rgba(255, 165, 0, 0.7); {{ '}' }}
.box5 {{ '{' }} background-color: hsl(300, 100%, 75%); {{ '}' }}

Output:

Color: red
Color: #00ff00
Color: rgb(0, 0, 255)
Color: rgba(255, 165, 0, 0.7)
Color: hsl(300, 100%, 75%)

Tips:

  • Use hex or RGB for precise color control
  • Use RGBA for transparency effects
  • Named colors are simple and readable
  • Use online color pickers to get color codes
