CSS Tables

Tables can be styled using CSS to enhance readability and layout. You can style borders, padding, background colors, and more.

Basic Table Styling:

table {{ '{' }}
  width: 100%;
  border-collapse: collapse;
{{ '}' }}

th, td {{ '{' }}
  border: 1px solid #ccc;
  padding: 10px;
  text-align: left;
{{ '}' }}

th {{ '{' }} background-color: #198754; color: white; {{ '}' }}
tr:nth-child(even) {{ '{' }} background-color: #f2f2f2; {{ '}' }}

Output:

Name Subject Score
Ravi HTML 85

Murari JavaScript 95

Tips:

  • Use border-collapse: collapse; to remove space between borders
  • Alternate row colors improve readability
  • th is used for table headers
  • Use padding and alignment for neatness
