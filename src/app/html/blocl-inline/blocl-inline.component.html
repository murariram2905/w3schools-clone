HTML Block vs Inline Elements

In HTML, elements are generally categorized as block-level or inline based on how they behave in the layout.

Block Elements:

  • Always start on a new line
  • Take up the full width available
  • Examples: <div>, <p>, <h1>â€“<h6>, <section>

Inline Elements:

  • Do not start on a new line
  • Only take up as much width as necessary
  • Examples: <span>, <a>, <strong>, <em>

Tip:

  • Block elements are useful for layout structure.
  • Inline elements are best for formatting content within lines.
  • You can change the display behavior using CSS display: block or display: inline.
