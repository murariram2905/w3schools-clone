HTML Comments

HTML comments are not displayed in the browser but can help explain your code or make it more readable.

Syntax:

<!-- This is a comment -->

Example:

<p>This is a paragraph.</p>
<!-- This is a comment that wonâ€™t be shown in the browser -->
<p>This is another paragraph.</p>

Output:

This is a paragraph.

This is another paragraph.

Notes:

  • Comments start with <!-- and end with -->.
  • They are useful for explaining code or temporarily disabling parts of HTML during testing.
  • Comments do not appear in the rendered output of the page.
