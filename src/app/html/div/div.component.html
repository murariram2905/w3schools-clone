HTML <div> Element

The <div> element is a block-level container used to group other HTML elements. It's commonly used for layout and styling.

Key Characteristics:

  • Block-level element
  • Used as a wrapper for sections or content blocks
  • Often styled with CSS for layout or design

Basic Example:

<div>
  <h2>Hello</h2>
  <p>This is inside a div.</p>
</div>

Rendered Output:

Hello

This is inside a div.

Use Case Example:

<div class="header">...</div>
<div class="main">...</div>
<div class="footer">...</div>

Notes:

  • By default, <div> elements stack vertically.
  • They do not apply any visual style unless styled with CSS.
  • Great for page structure and layout control using Flexbox or Grid.
