HTML Colors

In HTML, colors can be specified by using color names, HEX values, or RGB values.

Using Color Names:

This text is red.

<p style="color: red;">This text is red.</p>

Using HEX Values:

This text is green.

<p style="color: #00ff00;">This text is green.</p>

Using RGB Values:

This text is blue.

<p style="color: rgb(0, 0, 255);">This text is blue.</p>

Color Box Examples:

Red
#00ff00
rgb(0, 0, 255)

Notes:

  • You can use colors for text, backgrounds, borders, etc.
  • Common formats: name, hex, rgb, rgba, hsl
  • Always choose readable color combinations for accessibility.
