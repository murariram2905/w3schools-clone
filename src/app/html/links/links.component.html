HTML Links

HTML links are created using the <a> tag. The most important attribute of the <a> tag is href, which indicates the link's destination.

Basic Link:

Visit Google <a href="https://www.google.com" target="_blank">Visit Google</a>

Open Link in Same Tab:

Go to Example <a href="https://example.com">Go to Example</a>

Open Link in New Tab:

<a href="https://example.com" target="_blank">Open in new tab</a>

Email Link:

Send Email <a href="mailto:murariram2905@gmail.com">Send Email</a>

Phone Link:

Call Us <a href="tel:+911234567890">Call Us</a>

Notes:

  • href is used to define the URL.
  • target="_blank" opens the link in a new tab.
  • Use mailto: to send emails and tel: to call from mobile devices.
