HTML Paragraphs

Paragraphs in HTML are defined using the <p> tag. Browsers automatically add space before and after each paragraph.

Example:

This is a paragraph.

This is another paragraph.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Line Breaks:

You can use the <br> tag to insert a line break
like this.

<p>You can use the <br> tag to insert a line break <br> like this.</p>

Important Notes:

  • Paragraphs are block-level elements.
  • Never forget the closing </p> tag.
  • Multiple spaces and line breaks in code are ignored in browser output.
