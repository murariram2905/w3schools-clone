HTML Attributes

HTML attributes provide additional information about HTML elements. They are always specified in the start tag and usually come in name/value pairs like name="value".

Common HTML Attributes:

  • href – Specifies the URL for a link
  • src – Specifies the path to an image
  • alt – Provides alternative text for an image
  • style – Used to apply inline CSS styles
  • title – Shows a tooltip when hovering
  • id – Assigns a unique identifier
  • class – Assigns one or more class names
  • width and height – Specify dimensions for elements
